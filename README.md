
# Getting Started

## Prerequisites

 - Minikube
 - FaaS-CLI
 - Kubectl
 - Helm

## Installing and Deployment

 - Edit `/oidc-plugin/oidc-plugin-dep.yaml` to set your OpenID Connect configuration

 - Run the deployment script:

    > $ ./deploy.sh

 - Use the OpenFaaS Gateway: http://gw.secure-faas.com
